## Задание: PostgreSQL + Adminer в Docker

**Цель:** разработать и продемонстрировать локальное окружение для работы с СУБД PostgreSQL с использованием Docker и веб‑интерфейса Adminer.

---

### Требования к выполнению

#### 1. Развёртывание окружения

Создать конфигурацию на основе Docker (допускается использование Docker Compose), которая поднимает:

- контейнер с сервером **PostgreSQL**;
- контейнер с **Adminer**, доступный из браузера по адресу `http://localhost:<порт>`.

#### 2. Настройка подключения

Настроить подключение Adminer к развернутой базе данных PostgreSQL:

- обеспечить возможность входа через веб‑интерфейс;
- убедиться, что после авторизации отображается список баз данных и доступна работа с ними (создание таблиц, выполнение запросов и т.п.).

#### 3. Текстовое описание (README)

Подготовить краткое текстовое описание, включающее:

- используемые образы и порты;
- последовательность действий для запуска и остановки окружения;
- при наличии — основные проблемы, возникшие в процессе настройки, и способы их решения.

#### 4. Иллюстрации

Приложить:

- скриншот работающих контейнеров (вывод команды Docker, показывающий PostgreSQL и Adminer);
- скриншот страницы Adminer с успешно установленным подключением к PostgreSQL.
